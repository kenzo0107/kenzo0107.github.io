terraform はバージョンアップ頻度が高く、
プロジェクトによってバージョン差分が生じるので
複数バージョンを管理できると運用がスムーズです。

asdf 推した理由

terraform だけバージョン管理するのであれば tfenv がメジャーです。

ですが、 tflint, tfsec も導入を推奨しているので
あらゆるライブラリのバージョン管理が同じ方式でできるので
asdf を推してます。
