2015-12-24

pingを通すiptableの設定

環境

  • CentOS 5.8

実現したいこと

Server A が Server B からのみ ping を通す

特定IPからの ping を通す許可設定

以下設定を Server A で実施する。

  • --icmp-type 8 は Echo request (エコー要求) を許可
  • --icmp-type 0 は Echo Reply (エコー応答) を許可

Server Bから ping実行
