ディスク使用量が増加した際の調査方法

備忘録

ディスク使用量増加のアラートが上がったので調査した際の手順をまとめました。

調査法

以上の ①, ② の繰り返しによってどのディレクトリが高いか調査してます。

du の option は以下

  • -m : MB 表示
  • -s : 総計表示

普段は -h で見やすくしてますが、 今回の調査時には sort した際に MB や kB が混ざって表示され、直感的にわかり辛くなるので、 -ms にしてます。
